Glossary

This is my Glossary where I will be sharing a total of 50 Web Development terms used in the mordern day world of Internet Technology:

1.HTML: (HyperText Markup Language)

2.CMS: (Content Management System)

3.CSS: (Cascading Style Sheet)

4.PHP: (HyperText Preprocessor)

5.SEO: (Search Engine Optimisation)

6.Alt Attribute: (Used to describe an image)

7.Title attribute: (The text used to describe the webpage in the browser window)

8.Call to Action: (A button used to perform a specific action)

9.Promo Area: (Area on website used to promote products/services)

10.Hero: (This is the landing page)

11.UI: (Stands for 'User Interface' which is focused on the usability of the website)

12.UX: (Stands for 'User Experience')

13.Comment: (Bits of information which is stored in the HTML and used to identify parts of a file)

14.DHTML: (Dynamic HyperText Markup Language)

15.DOCTYPE: (DOCTYPE is written at the beginning of the html document to declare which version of HTML is used in a document)

16.XML: (Extensible Markup Language is mainly used to write custom markup languages.In addition, it allows multiple types of computerised devices to share information)

17. .HTACCESS: (.htacess file is the default directory-level configuration file)

18.Inheritance: (Elements which do not have a pre-defined style 'inherit' the style of the parent element)

19.Meta Data: (Meta Data is the data contained in the header that gives information about the web page that a user is on)

20.Meta Tag: (Meta tag is an HTML tag stated in the header of your webpage)

21.PermaLink: (This is for short 'Permanent Link')

22.Property: (Property is a CSS term and is similar to the HTML tag)

23.Pseudo-Element: (This is an element used to add special effect)

24.Pseudo Class: (similar to Pseudo-Element, Pseudo Class are used to add special effects)

25.Schema: (A Schema is an XML document used to describe other XML documents)

26.Script: (script is a portion of code on the HTML page that makes the page more interactive)

27.Selector: (In the CSS, the selector is the item which a style will be applied to)

28.Semantic Markup: (Semantic Markup refers to the use of tags such as 'header and paragraph')

29.SGML: ('Standard Generalised Markup Language' is a markup language used to organise theh structure of a document.)

30.TAG: (A tag is a set of markup charcters which are used alongside elements to indicate its start and end)

31.URL: ('Uniform Resource Locater' is a site's address which specifies where the page can be located on the internet.)

32.XHTML:(This stands for 'Extensible HyperText Markup Language' which has been developed to comply with XML syntax)

33.WEBPAGE: (A web page is a document which is written in a markup language such as HTML/XHTML)

34.WEB SERVER: (This is a software on computerised systems that hosts web sites and pages)

35.TEMPLATE: (A template is used to formulate a non-stop design across a website. Usually templates include structural information and give assistance on how a website should look)

36.SOAP: (This stands for 'Simple Objective Access Protocol' and it is a XML protocol that exchanges information across the internet to enable multiple application to access one another)

37.SERVER-SIDE: (Semantic Markup offers context to what a piece of content may contain)

38.RESOLUTION: (Refers to the number of pixels displayed on a screen. In the case of computers, display resolution does not refer to the number of pixels or dots of the computer screen, rather it is measured in inches

39.PROPERTY: (Property in CSS is similar to an HTML tag)

40.PROGRESSIVE ENHANCEMENT: (This is a method for web design that makes use of internet technologies in a way which allows users to access content and functionality of a page)

41.OPEN SOURCE:(Open source refers to the source code)

42.NON-BREAKING SPACE: (This is a white-space character which has the primary function of holding open table)

43.NESTING: (This term refers to assigning one HTML element with another. After this perfomed, the elements are closed in reverse order e.g. '<>')

44.NAVIGATION: (Navigation refers to the system that enables users to access multiple areas of a website)

45.LIQUID LAYOUT:(A liquid layout is focused on the browser windows size)

46.IMAGE MAP: (IMAGE map is used in XHTML to allow multiple parts of an image to be clickable)

47.HEXADECIMAL: (Hexadecimal is a base 16 numbering system used to distinguish colours online

48.FONT FAMILY: (Font family distinguishes the typefaces)

49.FOCAL POINT: (This is the part on the page on the web page where there could be a banner or some for of imagery to draw the reader'a attention

50.FAVCON: (Favicons are customisable icons)
