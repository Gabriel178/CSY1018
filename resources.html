RESOURCES

My resources page will contain all the most essential resources which the internet has to offer for learning coding. These will be based on HTML and CSS resources.

CODECADEMY:

An extremely platform to learn coding is with Codecademy. It has been used by 24 million people worldwide. Additionally, users are able to engage in courses that focus on the teaching of HTML/CSS, Javascript, Python and more.

UDEMY:

Udemy is an online platform which can be used to improve a user's coding skills. Furthermore, there are courses which are available to pay for, while there are also free options. HTML and CSS is also focucsed on with this platform.

Mozilla Thimble:

Mozilla Thimble is an online text editor which is available for free and enables a user to learn coding with HTML and CSS. It is often used in educational institutions as the user interface is user-friendly and simple to understand.

HTML5 Rocks:

This is a project launched by Google, which has many tutorials and is also an open-source platform. This coding platform is typically more advanced than other platforms, so it's adviced to gain some basic experience before playing with the tool.

CSS-TRICKS:

This platform used to be strically CSS based, but overtime it has now become a good platform to use as it incorparates all features of web development and web design.

CSSDECK:

CSS Deck allows you to code with other people in realtime, which can be done in multiple modes. This platform can also be shared on forums such as StackOverflow, where people can help solve any issues with code a user may have.

COURSERA:

Coursersa is a technology company that has over 1,000 courses available. There is also certification for completing particular programs.
